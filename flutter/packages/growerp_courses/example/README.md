# Courses Example

An example application for the GrowERP courses package.

## Features

This example demonstrates:
- Course management with AI-powered content generation
- Course modules and lessons organization
- Media generation for various platforms
- Course viewer with progress tracking

## Setup

It uses the local backend system from https://github.com/growerp/moqui

You can also use our test backend system:
    set in: example/assets/cfg/app_settings.json
        databaseUrlDebug: https://test.growerp.org

## Running the example

```bash
cd flutter/packages/growerp_courses/example
flutter run
```
